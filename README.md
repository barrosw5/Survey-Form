# Survey Form - freeCodeCamp

A web-based survey form designed to collect user feedback, developed as the first certification project for the freeCodeCamp Responsive Web Design course.

## Description

This project focuses on building a functional and accessible form using HTML5. It incorporates a variety of input types and validation techniques to ensure a structured data collection process. The form is designed to gather information from students and professionals about their experience with the freeCodeCamp platform.

Key technical concepts demonstrated in this repository include:
- Use of semantic HTML5 tags for form structure.
- Implementation of diverse input types: text, email, number, radio buttons, and checkboxes.
- Creation of dropdown selection menus.
- Form validation using 'required' attributes and specific input constraints (e.g., age range).
- Text area implementation for open-ended user feedback.

## Features

- User Identification: Fields for name and email with built-in validation.
- Age Constraint: Numeric input restricted to a specific age range (10-99).
- Role Selection: Dropdown menu to categorize the user's current professional status.
- Recommendation Tracking: Radio buttons for single-choice feedback.
- Improvement Checklist: Multiple checkboxes allowing users to select several areas for platform growth.
- Comments Section: Multi-line text input for detailed suggestions.

## Technologies Used

- HTML5: Used for the entire structure and form logic, ensuring accessibility and proper input handling.

## How to Use

1. Clone the repository to your local environment.
2. Open `index.html` in your preferred web browser.
3. Fill out the various sections of the form, including the required name and email fields.
4. Experiment with the validation by entering invalid data (e.g., an incorrect email format) to see the browser's native enforcement.
5. Click the "Submit" button to see the form's completion behavior.

## Project Structure

- `index.html`: The main document containing the survey form structure and all input elements.
